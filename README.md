# Java TCP Server (Networking)

## Overview
A Java TCP server that accepts client connections and handles message exchange.

## Key features
- Listen on a configurable port
- Accept client connections
- Handle messaging protocol (basic text)
- (Optional) Multi-client handling with threads

## Tech stack
- Java
- ServerSocket / Socket
- Multithreading (if used)

## How to run
1. Open project in Eclipse/IntelliJ
2. Run the server main class in `src/`
3. Server listens on port (e.g., 12345)
4. Run the client from `laptrinhmang` and connect to this server
